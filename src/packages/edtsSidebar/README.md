## EdtsSidebar

Sidebar and header shell widget for Mendix layouts. Keep the page body in the Mendix Layout `Main` placeholder; this widget only renders the custom sidebar and header chrome.

### Usage

- Place `EdtsSidebar` in the Layout.
- Drag a Mendix `Navigation tree`, `Menu bar`, or `Pop-up menu` into `Navigation Content`.
- Drag `EdtsProfileDropDown` or another widget into `Profile Content`.
- Keep the Layout `Main` placeholder outside this widget.

### Package
- Name: `edts-sidebar`
- Widget: `EdtsSidebar`

### Commands
- `npm run dev`
- `npm run build`
- `npm run lint`
